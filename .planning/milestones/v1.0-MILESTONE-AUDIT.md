## Tech Debt

### Phase 1: CLI and State Foundation
- **TypeScript error:** `simpleGit()` call produces TS2349 in `src/git/index.ts:7`. Runtime works (all 219 tests pass), but `tsc --noEmit` fails. Any CI type-checking pipeline will break.
- **Engine constraint:** `package.json` specifies `node >=20` but requirement CLI-04 says Node.js 18+. Either update the requirement or relax the engine constraint.

### Phase 3: Workflow Phases
- **Orphaned APIs:** `createPhase`, `updatePhaseStep`, `ensurePhaseDir` (from `src/phase/index.ts`) and `appendDecision`, `readDecisions`, `formatDecisionEntry` (from `src/phase/decisions.ts`) are exported but never called programmatically. Slash commands perform equivalent operations via direct file I/O. These functions could drift from actual behavior without detection.

### Phase 5: Team Coordination
- **Unused imports:** `warning` is imported but never called in `src/workstream/conflicts.ts` and `src/workstream/archive.ts`.
