<task type="auto" tdd="true">
  <name>Task 2: Core utilities - schema, output, constants</name>
  <files>src/constants.ts, src/state/schema.ts, src/output/index.ts, tests/state/schema.test.ts, tests/output/index.test.ts</files>
  <behavior>
    - CURRENT_SCHEMA_VERSION equals 1
    - migrateIfNeeded({ schemaVersion: 1, foo: "bar" }) returns object unchanged with schemaVersion 1
    - migrateIfNeeded({ foo: "bar" }) (missing schemaVersion) sets schemaVersion to 1
    - migrateIfNeeded preserves unknown fields (forward-compatible)
    - output({ key: "value" }, { json: true }) prints valid JSON to stdout
    - output({ key: "value" }, { json: false }) prints colored human-readable text
    - PROTECTED_BRANCHES includes "main", "master", "develop"
    - STRIP_PREFIXES includes "feature/", "fix/", "hotfix/", "bugfix/", "release/"
    - BRANCHOS_DIR equals ".branchos"
  </behavior>
  <action>
    1. Create `src/constants.ts`:
       - Export PROTECTED_BRANCHES: string[] = ['main', 'master', 'develop']
       - Export STRIP_PREFIXES: string[] = ['feature/', 'fix/', 'hotfix/', 'bugfix/', 'release/']
       - Export BRANCHOS_DIR = '.branchos'
       - Export SHARED_DIR = 'shared'
       - Export WORKSTREAMS_DIR = 'workstreams'
       - Export CONFIG_FILE = 'config.json'
       - Export RUNTIME_DIR = '.branchos-runtime'

    2. Create `src/state/schema.ts`:
       - Export CURRENT_SCHEMA_VERSION = 1
       - Export Migration interface: { fromVersion: number; migrate: (data: Record<string, unknown>) => Record<string, unknown> }
       - Export migrations: Migration[] = [] (empty for v1)
       - Export function migrateIfNeeded<T extends { schemaVersion: number }>(data: Record<string, unknown>): T
         - If schemaVersion missing, set to 0
         - Apply migrations in order where fromVersion matches current version
         - Set schemaVersion to CURRENT_SCHEMA_VERSION
         - Return as T
       - Loose validation: preserve all fields, only update schemaVersion

    3. Create `src/output/index.ts`:
       - Import chalk from 'chalk'
       - Export function output(data: Record<string, unknown>, options: { json?: boolean }): void
         - If options.json: console.log(JSON.stringify(data, null, 2))
         - Else: format key-value pairs with chalk colors (keys in cyan, values in white)
       - Export function error(message: string, options: { json?: boolean }): void
         - If options.json: console.error(JSON.stringify({ error: message }))
         - Else: console.error(chalk.red('Error: ') + message)
       - Export function success(message: string, options: { json?: boolean }): void
         - If options.json: console.log(JSON.stringify({ success: true, message }))
         - Else: console.log(chalk.green('✓ ') + message)

    4. Write tests FIRST (TDD red phase), then implement:
       - tests/state/schema.test.ts: test migrateIfNeeded with missing version, current version, unknown fields preserved
       - tests/output/index.test.ts: test JSON mode produces valid JSON, test human mode produces non-empty string (mock console.log)

    5. Run tests to confirm green.
  </action>
