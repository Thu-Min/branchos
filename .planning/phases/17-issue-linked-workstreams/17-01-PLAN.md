<objective>
Add issue-linked workstream creation via `create-workstream --issue #N`. The command fetches the GitHub issue, reverse-looks-up the corresponding feature, delegates to the existing feature-linked workstream flow, writes issue.md to the workstream directory, and surfaces issue context in context packets.

Purpose: Bridge GitHub issues to the workstream creation flow so developers can start work from an issue number instead of looking up the feature ID manually.
Output: fetchIssue helper, findFeatureByIssue reverse-lookup, issue-linked createWorkstream path, issue.md file format, and Issue Context section in context packets.
</objective>
